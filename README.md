# Ewidencja Pojazdów

Praca składa się z dwóch części, w ramach Etapów 1-4:

### 1\. Aplikacja Desktopowa (WinForms)

* Obsługa listy pojazdów w kontrolce DataGridView.
* Generowanie raportu RDLC (pojazdy z przeglądem w bieżącym miesiącu).
* Eksport danych do pliku PDF.

### 2\. Interfejs Webowy (Bootstrap 5)

* Responsywna tabela z listą pojazdów.
* Formularz dodawania otwierany w oknie modalnym.
* Kompletna walidacja pól z wykorzystaniem JavaScript.
